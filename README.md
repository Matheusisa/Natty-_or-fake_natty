# Vídeo: 40 segundos no Futuro das IAs Generativas 🎥

## 📒 Descrição
Este vídeo de 40 segundos apresenta uma visão rápida e envolvente do potencial e da aplicação das IAs Generativas na vida cotidiana.

## 🤖 Tecnologias Utilizadas
- IA Generativa **[Copilot da microsoft](https://copilot.microsoft.com/)** para roteirização e revisão;
- IA Generativa **[D-ID](https://www.d-id.com)** para sintetização de voz e geração de avatar virtual;
- IA Generativa **[Vidnoz](https://pt.vidnoz.com/clonar-voz.html)** para clonagem de voz ( Bob esponja );


## 🧐 Processo de Criação
Usei o "Copilot" para escrever e revisar o roteiro do vídeo. O "D-ID" foi essencial para criar um avatar virtual e voz sintetizada, trazendo uma dimensão futurística ao vídeo. E tambem o "Vidnoz" para clonagem de voz, que no caso foi a do personagem Bob esponja.

## 🚀 Resultados
O vídeo é uma cápsula do tempo que leva o espectador a uma jornada rápida pelo mundo das IAs Generativas, destacando seu potencial transformador.
